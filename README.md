# 启学教育官网重设计

这是一个面向教育培训机构的静态官网首页重设计方案，适合后续放到 GitHub 后用 Codespaces 继续开发。

## 技术结构

- `index.html`：首页结构
- `styles.css`：页面样式
- `script.js`：导航与表单交互

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

## 云上开发建议

1. 把当前目录推到 GitHub 仓库
2. 用 GitHub Codespaces 打开仓库
3. 在 Codespaces 中继续迭代页面
4. 最终部署到 GitHub Pages、Netlify 或 Vercel
